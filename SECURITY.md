# Security Policy

If you discover a security issue related to this repository, please report it responsibly.

This project focuses on documentation and knowledge about technical support and product improvement.

## Reporting a Vulnerability

If you believe there is a security issue related to the content or examples in this repository, please open a private discussion or contact the maintainer.

Do not disclose security vulnerabilities publicly before they have been reviewed.

Thank you for helping improve the safety of this project.
